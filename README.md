# Projeto de Pesquisa — Linguagem Java

## Sobre o Projeto

Este repositório foi desenvolvido como atividade da disciplina **Tópicos Essenciais para Programação** do curso de **Ciência da Computação — UNIESP**.

O objetivo do projeto é pesquisar a linguagem de programação **Java**, organizando as informações utilizando práticas de metodologia ágil, além de acompanhar o progresso das tarefas da equipe.

---

## Objetivos da Pesquisa

- Conhecer a história da linguagem Java
- Identificar seus criadores
- Explorar as principais características
- Apresentar exemplos de código
- Mostrar áreas de aplicação
- Analisar vantagens e desvantagens
- Pesquisar empresas que utilizam Java
- Apresentar curiosidades sobre a linguagem
